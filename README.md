A modern Android Wishlist application built using **Kotlin + Jetpack Compose**.  
It allows users to add, update, and delete wishlist items with persistent local storage.

---

## 📱 Overview

WishlistApp is a simple yet structured Android application that helps users keep track of items they plan to purchase.

Users can:
- Add a wishlist item with title and description
- Edit existing items
- Delete items
- View all saved items in a clean Compose UI
- Store data locally using Room Database

## 🧠 What I Learned

This project was not just about building a wishlist — it was about understanding proper Android architecture.

### 🔹 MVVM Architecture
- Separation of concerns (UI, Business Logic, Data)
- ViewModel for managing UI-related data
- Observing state changes in Compose
- Cleaner and scalable code structure

### 🔹 Room Database
- Creating Entities, DAO, and Database classes
- Performing CRUD operations
- Understanding abstraction over SQLite
- Integrating Room with ViewModel

### 🔹 Kotlin Coroutines
- Running database operations off the main thread
- Using `suspend` functions
- Understanding asynchronous programming
- Managing structured concurrency in Android

### 🔹 Jetpack Compose
- State management
- Recomposition and UI updates
- Building modern declarative UI instead of XML
- Creating reusable composables
